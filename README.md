# بازی معروف سنگ کاغذ قیچی (ساخته شده با هوش مصنوعی) 🤖
بازی آنلاین نوستالژی سنگ کاغذ قیچی که با طراحی مهدی قاسمی بوده و با برنامه نویسی هوش مصنوعی بوده است این بازی رو چه آنلاین یا چه آفلاین می توانید بازی کنید که هر مقدار که سکه می گیرید می توانید پوسته خود رو تغییر دهید یا حتا عنوان های حماسی از پوسته کلاسیک گرفته تا پوسته هولوگرام هوشمند از نوآموز سنگ گرفته تا امپراطور سنگ کاغذ قیچی این بازی می تواتید بازی کنید چه به صورت محلی چه به صورت آنلاین با دوستان خود بازی کنید امیدوارم از این بازی خوشتان بیاد
# نحوه اجرا کردن ◀️
اول پروژه کلون کنید بعد اجرا کنید:
```bash
git clone https://github.com/mahdiprogrammer1/rps-game-online-
cd rps-game-online-
npm start
```
